# linkshortify-autoskip

> A ViolentMonkey userscript that automatically skips the LinkShortify (lksfy.com) countdown and spoofs Chrome browser identity for verification pages — fully hands free.

---

## What it does

- Automatically waits and clicks through the LinkShortify countdown timer
- Spoofs Chrome browser identity so verification pages don't block you
- Works in Firefox Android (with Cloudflare DNS) and Kiwi Browser

---

## Requirements

- [Firefox for Android](https://play.google.com/store/apps/details?id=org.mozilla.firefox) with Cloudflare DNS **or** [Kiwi Browser](https://play.google.com/store/apps/details?id=com.kiwibrowser.browser)
- [ViolentMonkey](https://addons.mozilla.org/en-US/firefox/addon/violentmonkey/) extension installed in your browser
- [uBlock Origin](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/) *(optional but recommended to block ads)*

---

## Installation

1. Open your browser on Android
2. Make sure **ViolentMonkey** is installed
3. Tap the install link for your preferred version below
4. ViolentMonkey will prompt you to install automatically

3. Done
